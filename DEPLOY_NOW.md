   - Port: `5432`
   - Database: `lms_db`

  production: true,
  apiBaseUrl: "https://lms-backend-xxxx.onrender.com/api", // ← Thay URL backend của bạn

## 🐛 Troubleshooting

### ❌ Backend 503 Service Unavailable

**Nguyên nhân:** Render free tier sleep sau 15 phút không hoạt động

**Giải pháp:**

1. Đợi 30-60 giây để backend "wake up"
2. Refresh lại page
3. Hoặc ping trước: `curl https://lms-backend-xxxx.onrender.com/actuator/health`

### ❌ Frontend không connect được backend (CORS error)

**Kiểm tra:**

```powershell
# 1. Backend có cho phép origin chưa?
curl https://lms-backend-xxxx.onrender.com/actuator/env | grep CORS

# 2. Frontend API URL đúng chưa?
# Check: lms-frontend/src/environments/environment.prod.ts
```

**Fix:**

- Update `CORS_ALLOWED_ORIGINS` trong Render Environment
- Đảm bảo `apiBaseUrl` trong `environment.prod.ts` đúng
- Redeploy backend (Save Changes trong Render)

### ❌ Database connection error

**Kiểm tra:**

```bash
# 1. DB_URL có đúng format không?
DB_URL=jdbc:postgresql://host:5432/lms_db

# 2. Username/Password đúng chưa?
# Copy từ Render Database dashboard

# 3. Database có data chưa?
# Import lại: psql -h ... -f lms_db_backup.sql
```

### ❌ Build failed on Render

**Nguyên nhân:** Docker build lỗi

**Giải pháp:**

1. Check logs trong Render Dashboard → Logs tab
2. Thường là Maven dependencies timeout
3. Retry deploy: Click "Manual Deploy" → "Deploy latest commit"
